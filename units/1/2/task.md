# 1.2 задача

## Условие

Переведите следующее выражение в префиксную форму

```python
(5 + 4 + (2 - (3 - (6 + (4/5))))) / (3 * (6 - 2) * (2 - 7))
```

## Решение

```scheme
(/ (+ 5
      4
      (- 2
         (- 3
            (+ 6
               (/ 4 5))))
   (* 3
      (- 6 2)
      (- 2 7))))
```

```scheme
(define (expmod base exp m)

```

Вместо старого алгоритма expmod для вычисления степени числа по модулю другого числа:

```scheme
(define (expmod base exp m)
   (cond ((= exp 0) 1)
         ((even? exp) (remainder (square (expmod base (/ exp 2) m)) m))
         (else (remainder (* base (expmod base (- exp 1) m)) m))))
```
